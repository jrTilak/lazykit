## Props

- `value` (`T`) — Value to throttle.
- `intervalMs` (`number`) — Finite, non-negative minimum interval between published changes.

### Returns

`T` — The initial value immediately and the latest pending value when the current throttle window ends. Function values are stored as values and are never invoked by the hook.
