## Props

- `initialItems` (`readonly Item[]`, default: `[]`) — Dense initial list and
  latest reset snapshot. Sparse arrays are rejected.

### Returns

`UseListResult<Item>` with readonly `items`, `setItems`, `append`, `prepend`,
`insertAt`, `setAt`, `updateAt`, `move`, `removeAt`, `removeWhere`, `clear`, and
`reset`. Index controls are no-ops for non-safe or out-of-range indexes.
