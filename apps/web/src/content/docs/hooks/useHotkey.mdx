## Props

- `hotkeys` (`string | readonly string[]`) — Shortcuts such as `"mod+s"` or
  `"shift+alt+k"`. Each chord must contain exactly one non-modifier key. Use
  `"plus"` for the `+` key, such as `"shift+plus"`.
- `handler` — Receives the matching keyboard event.
- `options` — Controls target, enabled state, repeats, editable elements,
  exact modifiers, and default prevention.

### Returns

Nothing. `mod` accepts either Control or Command. With exact modifiers enabled,
pressing both does not match.
