## Props

- `stepCount` (`number`) — Positive safe integer number of zero-based steps.
- `options.initialStep` (`number`, default: `0`) — Initial and reset step.
- `options.loop` (`boolean`, default: `false`) — Wraps past either boundary
  instead of clamping.

### Returns

`UseStepResult` containing `currentStep`, React-compatible `setStep`, `next`,
`previous`, `reset`, `isFirst`, and `isLast`. Steps are normalized again if
`stepCount` changes.
