## Props

- `ref` (`RefObject<ElementType | null>`) — Optional fullscreen target. The
  document root is used only when the argument is omitted; a supplied empty ref
  is not redirected to the root.

### Returns

`UseFullscreenReturn` with state, support, error, and async `enter`, `exit`, and
`toggle` controls. Controls resolve `false` when unavailable or rejected.
