## Props

- `initialState` (`State`) — Plain object used on mount and captured again as
  the latest reset snapshot. Arrays, functions, and class instances are not
  supported.

### Returns

`UseObjectStateResult<State>` with `state`, full `setState`, shallow `patch`,
generic `setKey` and `updateKey`, and `reset`. Copies preserve the plain
object's prototype and every own property descriptor, including symbols,
non-enumerable fields, accessors, and an own `__proto__` key.
