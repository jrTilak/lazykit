## Props

- `callback` (`(this: T, ...args: A) => unknown`) — Function invoked with the receiver and arguments from the latest call.
- `delayMs` (`number`, default: `300`) — Finite, non-negative quiet period in milliseconds.

### Returns

A receiver- and argument-safe debounced function with:

- `cancel()` — Discards a pending invocation.
- `flush()` — Immediately runs a pending invocation.
- `pending()` — Reports whether an invocation is waiting.

The returned function keeps the same identity across renders. Changing the delay or unmounting cancels pending work.
