## Props

- `callback` (`Callback`) — Function to invoke. Its complete callable type,
  including generic or overloaded signatures and an explicit receiver, is
  preserved.

### Returns

`EventCallback<Callback>` — A callback with stable identity. After each commit,
it invokes the latest `callback` supplied to the hook.
