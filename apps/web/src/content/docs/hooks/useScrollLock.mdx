## Props

- `ref` — Optional element ref. Omitting it selects the document body.
- `options.enabled` — Enables the lock. Defaults to `true`.
- `options.preserveScrollbarGap` — Adds body padding for the removed scrollbar.
  The gap remains while any nested lock requests it.

### Returns

Nothing. Nested hook instances are reference-counted and original styles are
restored after the final lock releases.
