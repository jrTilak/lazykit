## Props

- `initialEntries` (`ReadonlyMap<Key, Value> | readonly [Key, Value][]`,
  default: `[]`) — Initial entries and latest reset snapshot.

### Returns

`UseMapResult<Key, Value>` with readonly `map`, `set`, `setMany`, `update`,
`remove`, `clear`, and `reset`. `setMany` materializes an iterable once before
the React state update, so one-shot generators are safe. The `update`
callback receives both the current value and an `exists` flag, distinguishing
a missing key from a key whose value is `undefined`.
