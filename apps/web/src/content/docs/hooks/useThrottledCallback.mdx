## Props

- `callback` (`(this: T, ...args: A) => R`) — Latest callback to invoke with its original receiver and arguments.
- `intervalMs` (`number`) — Finite, non-negative interval in milliseconds.

### Returns

A stable leading-edge function returning `R | undefined`. Accepted calls return the callback result; dropped calls return `undefined`. `reset()` and an `intervalMs` change immediately open the throttle window.
