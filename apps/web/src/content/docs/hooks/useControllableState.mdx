## Props

- `options.defaultValue` (`Value`) — Initial value when the `value` property is
  omitted.
- `options.value` (`Value`, optional) — Controlled value. An explicitly present
  `undefined` still selects controlled mode.
- `options.onChange` (`(value: Value) => void`, optional) — Receives requested
  changes in either mode.

### Returns

`UseControllableStateResult<Value>` with `value`, stable React-compatible
`setValue`, and `isControlled`. Controlled updates notify `onChange` without
changing the supplied value. Switching to uncontrolled mode starts from the
last committed controlled value. Replace callable values through an updater
such as `setValue(() => nextFunction)`.
