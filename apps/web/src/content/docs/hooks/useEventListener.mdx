## Props

- `target` (`EventListenerTarget<Target>`) — An event target, ref, or `null`.
- `type` — Event name inferred from the target's event map.
- `handler` — Receives the matching typed event.
- `options` — Native listener options plus `enabled`.

### Returns

Nothing. The listener follows ref target swaps and is removed on cleanup.
