## Props

- `title` (`string | null`) — Title owned by this hook instance. `null` removes its current title layer.
- `options.restoreOnUnmount` (`boolean`, default: `true`) — Removes this instance's title layer on unmount. `false` retains that layer.

### Returns

`void`. The most recently updated mounted instance owns the visible title. Removing instances out of order reveals the latest remaining layer. The hook performs no document access during server rendering.
