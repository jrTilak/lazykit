## Props

- `value` (`T`) — Value to publish after it settles.
- `delayMs` (`number`, default: `300`) — Finite, non-negative delay in milliseconds.

### Returns

`T` — The initial value immediately, then the latest value whose delay completed. A value or delay change replaces the pending schedule. Function values are stored as values and are never invoked by the hook.
