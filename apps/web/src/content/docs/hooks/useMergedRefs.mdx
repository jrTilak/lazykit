## Props

- `refs` (`readonly (Ref<Value> | null | undefined)[]`) — Callback or object
  refs that should all receive the same value.

### Returns

`RefCallback<Value>` — A merged callback that assigns the mounted value and
`null` cleanup to every supplied ref. Its identity stays stable when refs are
added, removed, or reordered; newly added refs receive the current value and
removed refs receive `null` after commit.
