## Props

- `initialValue` (`number`, default: `0`) — Initial and reset value.
- `options.min` (`number`, optional) — Inclusive lower bound.
- `options.max` (`number`, optional) — Inclusive upper bound.
- `options.step` (`number`, default: `1`) — Finite positive default increment
  and decrement amount.

All numbers must be finite and `min` cannot exceed `max`.

### Returns

`UseCounterResult` with `count`, bounded `setCount`, `increment`, `decrement`,
and `reset`. Passing a finite non-negative amount to `increment` or `decrement`
overrides `step` for that call. Use `setCount` for signed arithmetic. The
current count is clamped again when either bound changes.
