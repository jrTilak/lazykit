## Props

- `initialValues` (`ReadonlySet<Value> | readonly Value[]`, default: `[]`) —
  Initial values and latest reset snapshot. Sparse arrays are rejected, while
  an explicit `undefined` remains a valid value when included in `Value`.

### Returns

`UseSetResult<Value>` with readonly `set`, `add`, `addMany`, `remove`, `toggle`,
`clear`, and `reset`. Duplicate additions and missing removals retain the
existing Set identity.
