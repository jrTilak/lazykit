## Props

- `input` (`string`) — The display label to canonicalize.

### Returns

`DisplayName`, a `string` branded with `"DisplayName"`. Outer whitespace is
trimmed, Unicode text is normalized to NFC, and the result must contain 1 to 80
Unicode code points. Unpaired UTF-16 surrogates are rejected.

Ordinary internal whitespace is preserved. C0 and C1 control characters,
including internal tabs and line breaks, are rejected after outer whitespace is
trimmed. The schema deliberately permits all scripts, emoji, apostrophes,
hyphens, and other well-formed characters; it validates a display label, not a
person's legal or culturally specific name.
