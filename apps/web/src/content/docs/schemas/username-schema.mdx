## Props

- `input` (`string`) — The candidate username.

### Returns

`Username`, a canonical `string` branded with `"Username"`. The schema trims
outer whitespace, lowercases the value, normalizes it to Unicode NFC, and then
requires 3 to 30 Unicode code points.

The final value contains lowercase ASCII letters or numbers separated by single
dots, underscores, or hyphens. Separators cannot be repeated or appear at
either edge. Availability and reserved-name checks remain application-level
concerns.
