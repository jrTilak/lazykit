## Props

- `input` (`string`) — The candidate email address. Leading and trailing
  whitespace is removed before validation.

### Returns

`Email`, a `string` branded with `"Email"`. The trimmed value must pass Zod 4's
email format and contain at most 254 characters.

Letter casing is preserved, including the local part. The schema validates
syntax only; it does not confirm that the address or mailbox exists.
