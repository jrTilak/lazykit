## Props

- `fn` (`(...args) => Return`) — Function whose successful results are cached.
- The invocation receiver and object arguments use reference identity; primitive keys use `Map` equality.
- The returned function provides `clear()`.
- Properties on a callable object remain available through the wrapper.
- A callable that already defines `clear` is rejected to avoid a control-name collision.
