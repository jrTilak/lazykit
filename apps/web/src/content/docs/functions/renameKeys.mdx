## Props

- `object` (`T`) — Object to copy.
- `mappings` (`readonly { from: keyof T; to: PropertyKey }[]`) — Source and destination key pairs. Values are read from the original object, so mappings do not chain through one another.

### Returns

A new object with each existing source key replaced by its destination key. When destinations collide, the last mapping wins.
