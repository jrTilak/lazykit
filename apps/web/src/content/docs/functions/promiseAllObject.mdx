## Props

- `object` (`PromiseAllObjectInput<T>`) — A plain object containing values or promises.
- Returns `Promise<PromiseAllObjectResult<T>>`; readonly properties become writable and each value is recursively unwrapped with `Awaited`.
- All own string and symbol properties are included, including non-enumerable properties.
- The resolved result preserves the source's ordinary or null prototype.
- Arrays, functions, class instances, and objects with custom prototypes reject with `TypeError`.
- A `then` property may be returned when it is not callable. Callable resolved `then` values reject to prevent Promise thenable assimilation.
