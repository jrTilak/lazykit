## Props

- `array` (`readonly T[]`) — Values to divide.
- `predicate` (`(value, index, array) => boolean`) — Returns `true` for the first group and `false` for the second.

### Returns

`[T[], T[]]` — Matching values followed by non-matching values, with order preserved in both groups.
