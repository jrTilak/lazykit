## Props

- `fn` (`(...args: A) => void`) — Function invoked with the arguments from the latest call.
- `delayMs` (`number`, default: `300`) — Finite, non-negative quiet period in milliseconds.

### Returns

A debounced function with `cancel()`, `flush()`, and `pending()` controls.
