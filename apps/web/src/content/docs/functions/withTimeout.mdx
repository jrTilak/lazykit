## Props

- `fn` (`(...args: A) => R | PromiseLike<R>`) — Work to invoke with the wrapper’s arguments.
- `timeoutMs` (`number`) — Finite, non-negative deadline in milliseconds.
- `createError` (`(...args: A) => Error`) — Optional factory for a contextual timeout error.

### Returns

`(...args: A) => Promise<Awaited<R>>` — A wrapper that resolves with the operation or rejects with its error or the timeout error.
