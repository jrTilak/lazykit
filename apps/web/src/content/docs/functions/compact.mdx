## Props

- `array` (`readonly T[]`) — Values to filter. JavaScript falsy values such as `false`, `0`, `0n`, `""`, `null`, `undefined`, and `NaN` are removed.

### Returns

`Array<Exclude<T, Falsy>>` — A new array containing the truthy values.
