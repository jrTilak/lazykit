## Props

- `object` (`T`) — Non-array object whose own enumerable values should be transformed.
- `transform` (`(value, key, object) => U`) — Produces the replacement value for each property and is invoked without a receiver. Numeric TypeScript keys arrive as their runtime string form.

### Returns

`Partial<MappedObject<T, U>>` — A new null-prototype object containing the
transformed own enumerable keys. Properties are optional in the return type
because TypeScript cannot identify non-enumerable properties statically.

Arrays are rejected with `TypeError`. Symbol and special keys such as
`__proto__` are preserved as safe own data properties.
