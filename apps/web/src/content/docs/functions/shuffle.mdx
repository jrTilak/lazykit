## Props

- `array` (`readonly T[]`) — Values to shuffle.
- `random` (`() => number`, default: `Math.random`) — Random source returning a number from `0` up to, but not including, `1`. Supply a seeded source for reproducible output.

### Returns

`T[]` — A shuffled copy. The input is not mutated.
