---


<CodePreview methodName="useIsOnline"/>

## Installtion

```bash
npx @jrtilak/lazykit add useIsOnline
```

## Returns

- `isOnline`: A boolean value that represents the online status of the user. It returns `true` if the user is online, and `false` if the user is offline.
