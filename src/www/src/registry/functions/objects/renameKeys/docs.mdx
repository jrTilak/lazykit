---
name: renameKeys
desc: Renames keys in an object.
---

<CodePreview methodName="renameKeys"/>

## Installtion

```bash
npx @jrtilak/lazykit add renameKeys
```
## Parameters

- `obj` (`T`)  
  The input object whose keys will be renamed.

- `keys` (`Array<{ old: K; new: N }>`)
  An array of objects specifying which keys to rename. Each object has two properties:
  - `old` (`K`): The key in the original object to be renamed.
  - `new` (`N`): The new key name.

## Returns

- `Omit<T, K> & Record<N, T[K]>`  
  A new object with the specified keys renamed.

### Example Usage

```javascript
"use js-executor";

import renameKeys from "@/helpers/renameKeys";

const user = {
  firstName: "Alice",
  lastName: "Smith",
  age: 28,
};

const renamedUser = renameKeys(user, [
  { old: "firstName", new: "first_name" },
  { old: "lastName", new: "last_name" },
]);

console.log(renamedUser); 
// Output: { age: 28, first_name: "Alice", last_name: "Smith" }
```

### Example with Nested Object

```javascript
"use js-executor";

import renameKeys from "@/helpers/renameKeys";

const product = {
  productId: 101,
  productName: "Laptop",
  productPrice: 1200,
};

const renamedProduct = renameKeys(product, [
  { old: "productId", new: "id" },
  { old: "productName", new: "name" },
]);

console.log(renamedProduct); 
// Output: { productPrice: 1200, id: 101, name: "Laptop" }
```
