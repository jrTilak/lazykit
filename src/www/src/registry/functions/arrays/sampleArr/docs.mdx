## Usage
### Positive Index
```javascript
"use js-executor"
import sampleArr from "@/helpers/sampleArr";

const arr = sampleArr(5);
console.log(arr);
// Expected output: an array of 5 random numbers
```
### Negative Index

```javascript
"use js-executor"
import sampleArr from "@/helpers/sampleArr";

const arr = sampleArr(-5);
console.log(arr);
// Expected output: Error
```
