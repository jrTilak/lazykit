name: partition
desc: Partitions an array into two arrays based on a given callback ie predicate.
---


<CodePreview methodName="partition"/>

## Installtion

```bash
npx @jrtilak/lazykit add partition
```

## Props
<PropsTable methodName="partition"/>
---

- `array`: (required) The array to partition into two groups.
  
- `predicate`: (required) A function that determines the partitioning; should return `true` for elements in the first group and `false` for elements in the second.

### Usage

### Partitioning even and odd numbers
```javascript
"use js-executor";

import partition from '@/helpers/partition';

const numbers = [1, 2, 3, 4, 5, 6];
const [evens, odds] = partition(numbers, (num) => num % 2 === 0);

console.log(evens); // Output: [2, 4, 6]
console.log(odds);  // Output: [1, 3, 5]
```
### Partitioning by object property
```javascript
"use js-executor";
import partition from '@/helpers/partition';

const people = [
  { name: 'Alice', active: true },
  { name: 'Bob', active: false },
  { name: 'Charlie', active: true },
];

const [active, inactive] = partition(people, (person) => person.active);

console.log(active);   // Output: [{ name: 'Alice', active: true }, { name: 'Charlie', active: true }]
console.log(inactive); // Output: [{ name: 'Bob', active: false }]
```
### Partition strings by length
```javascript
"use js-executor";
import partition from '@/helpers/partition';

const words = ["short", "exceedingly long", "medium", "tiny"];
const [longWords, shortWords] = partition(words, (word) => word.length > 5);

console.log(longWords);  // Output: ["exceedingly long", "medium"]
console.log(shortWords); // Output: ["short", "tiny"]
```
