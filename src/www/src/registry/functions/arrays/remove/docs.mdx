---
name: remove
desc: Removes elements from an array at a specified index.
---

<CodePreview methodName="remove"/>

## Installtion

```bash
npx @jrtilak/lazykit add remove
```

## Props
<PropsTable methodName="remove"/>
---

- `array`: (required) The array to process and remove elements from.

- `index(s)`: (required) The index (or indices) of elements to remove from the array. Negative indices are supported, representing positions counted from the end of the array.

## Example Usage

### Remove a single element by index
```javascript
"use js-executor";

import remove from '@/helpers/remove';

const numbers = [10, 20, 30, 40, 50];
const result1 = remove(numbers, 2);

console.log(result1); 
// Output: [10, 20, 40, 50]
```

### Remove multiple elements by indices
```javascript
"use js-executor";
import remove from '@/helpers/remove';

const numbers = [10, 20, 30, 40, 50];
const result2 = remove(numbers, [1, 3]);

console.log(result2); 
// Output: [10, 30, 50]
```
### Using negative indices
```javascript
"use js-executor";
import remove from '@/helpers/remove';

const numbers = [10, 20, 30, 40, 50];
const result3 = remove(numbers, [-1, -3]);

console.log(result3); 
// Output: [10, 20, 40]
```
