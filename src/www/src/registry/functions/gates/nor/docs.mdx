---
name: nor
desc: Performs a logical NOR operation on the given arguments.
---


<CodePreview methodName="nor"/>

## Installtion

```bash
npx @jrtilak/lazykit add nor
```

## Parameters

- `...args` (`unknown[]`)  
  A variable number of arguments to evaluate in the logical NOR operation.

## Returns

- `boolean`  
  Returns `true` if all arguments are falsy; otherwise, returns `false`.


## Usage
```javascript
"use js-executor"
import nor from "@/helpers/nor";

console.log(nor(true, true));
// Expected Output: false

console.log(nor(true, false));
// Expected Output: false

console.log(nor(false, false));
// Expected Output: true

console.log(nor());
// Expected Output: false

console.log(nor(1, "lazykit"));
// Expected Output: false
```
