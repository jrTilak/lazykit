---
name: deepCompare
desc:  compares two or more values deeply, checking both primitive and non-primitive types for equality by comparing their serialized JSON string representations.
---


<CodePreview methodName="deepCompare"/>

## Installtion

```bash
npx @jrtilak/lazykit add deepCompare
```

## Parameters

- `a` (`unknown`): The first value to compare.
- `b` (`unknown`): The second value to compare.

## Returns

- `boolean`: `true` if the values are deeply equal, otherwise `false`.

## Example Usage

```js
"use js-executor"
import deepCompare from '@/helpers/deepCompare';

const obj1 = { name: 'Alice', age: 30 };
const obj2 = { name: 'Alice', age: 30 };
const obj3 = { name: 'Bob', age: 25 };

console.log(deepCompare(obj1, obj2)); // true
console.log(deepCompare(obj1, obj3)); // false
console.log(deepCompare(10, 10));     // true
console.log(deepCompare('hello', 'hello', obj1)); // false
```
